# CTF {blockchain} challenge

## How to install

### Install depedencies

```shell
npm install
```

### Compile the solidity

1. delete `./artifacts`
2. delete `./build`
3. delete `./cache`

and run this command below:

```shell
npm run compile
```

### Fork the Binance blockchain

```shell
npm run hh:deploy
```

### Find the flag

Modify the deploy scripts to get the flag
